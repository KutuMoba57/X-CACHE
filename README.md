


**X-CACHE V6.3**

`⦿` clean application cache and useless files from the system to reduce internal memory usage 



**FEATURE :**

`❖` *auto cleans in :*

   *⊙* `10 minutes after screen off.`

   *⊙* `30 minutes after screen off.`

   *⊙* `60 minutes after screen off.`

`❖` *can be automatic, can be manual.*



**INSTALLATION :**

`⊙` flash

`⊙` select mode

`⊙` reboot system

`⊙` enjoy 



**NOTES:**

`❖` Not only cache, other junk files will also be lost, both those that have just appeared and those that have been around for a long time.

`❖` If you want to clean manually, access su -c xcache in the terminal.
